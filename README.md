# LifeSync

## Sobre o projeto

O projeto trata-se de um app desenvolvido para dispositivos mobile e desktop, onde seu principal objetivo é que o usuário consiga organizar áreas da sua vida, como por exemplo, suas finanças, metas, notas e tarefas. Todos distribuídos em áreas separadas, específicas para cada uma delas.
